# 校园 AI 大赛产品试映舱改造报告

## 结果

已将当前默认产品从“发布前社媒试映”调整为面向腾讯 PCG 校园 AI 产品创意大赛的“参赛提交前试映舱”。

## 需求来源

- 需求图片目录：`D:\pcg-AIproduct\_wx_imgs`
- 识别到的核心要求：赛程日历、5 大命题 + 1 道开放题、参赛对象、奖励信息、提交三件套、报名时间和咨询入口。

## 改动范围

- `src/pages/PreflightStudioPage.tsx`：改造默认场景、Demo 示例、表单字段、赛事信息区、结果标题和空状态。
- `src/styles/pages.css`：补充赛事信息区样式，修正桌面和移动端响应式约束。
- `shared/preflightSimulation.ts`：新增 `campus_ai_competition` 评审场景和 `submission_readiness` 验证目标。
- `server/lib/preflight/mockProvider.ts`：新增大赛评审语义的本地 mock 结果。
- `server/lib/preflight/prompt.ts`：按大赛提交前检查重写模型提示分支。
- `server/lib/preflight/preflightJobStore.ts`、`server/lib/preflight/runPreflightSimulation.ts`：将阶段标签改为提交前反馈语义。
- `src/pages/PreflightStudioPage.test.tsx`：更新页面定位测试。
