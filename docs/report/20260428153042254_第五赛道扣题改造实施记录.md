# 第五赛道扣题改造实施记录

## 结果

已完成第五赛道 P0 扣题改造：默认产品叙事收敛为 `KOC Growth Lab`，新增结构化 `growthBrief` 输出，并在结果区显式呈现“内容方向、选题建议、发布策略、互动优化、账号成长、发布风险护栏”。

## 改动点

- `shared/preflightSimulation.ts`
  - 新增 `PreflightGrowthBrief` 结果结构。
  - `PreflightSimulationResult` 增加 `growthBrief`。
  - parser 支持真实 provider 原生 `growthBrief`，也能从内容承诺、视觉读片、评论、动作和安全审查中生成 fallback。

- `server/lib/preflight/prompt.ts`
  - JSON contract 增加 `growthBrief`。
  - prompt 明确要求输出第五赛道关键词：内容方向、选题建议、发布策略、互动优化、账号成长、发布风险护栏。

- `server/lib/preflight/mockProvider.ts`
  - mock provider 增加第五赛道专用 `growthBrief`。
  - 默认 KOC 场景强化为“99 元宿舍桌面改造”的内容方向、下一条选题、发布策略、评论互动和账号成长计划。

- `server/lib/preflight/runPreflightSimulation.ts`
  - 增长场景 synthesis 进度文案改为收拢内容方向、选题建议、发布策略、互动优化和账号成长计划。

- `src/pages/PreflightStudioPage.tsx`
  - 默认下拉隐藏 `campus_ai_competition` 和 `submission_readiness`，避免评委第一眼看到校园提交检查分支。
  - 首屏副标题改为第五赛道 KOC 社媒 AI Agent 叙事。
  - 新增 `GrowthBriefPanel`，结果区显式展示第五赛道扣题链路。
  - “发布安全审查”在 KOC 场景下改口径为“发布风险护栏”。

- `src/styles/pages.css`
  - 增加第五赛道扣题链路卡片和三条后续内容计划样式。

- `README.md`
  - 主标题改为 `KOC Growth Lab`。
  - 首段明确第五赛道、普通 KOC、社媒 AI Agent、涨粉。
  - 默认 demo 改述为学生 KOC 的小红书“99 元宿舍桌面改造”。

- `README.en.md`
  - 重写为英文版 KOC Growth Lab 项目说明，不再沿用旧 DeltaArc 游戏设计沙盘叙事。

- 测试
  - `shared/preflightSimulation.test.ts` 增加原生和 legacy fallback 的 `growthBrief` 断言。
  - `server/lib/preflight/runPreflightSimulation.test.ts` 增加远端 fallback 下 `growthBrief` 断言。
  - `src/pages/PreflightStudioPage.test.tsx` 更新第五赛道文案断言。

## 风险与回滚

- 校园提交检查分支没有删除，只是在默认下拉中隐藏；如果后续还要内部自检，可通过测试 fixture 或代码入口继续使用。
- `growthBrief` 对真实 provider 是新增字段；parser 已提供 fallback，不会因为模型漏字段导致页面空缺。
- 如果评委演示需要更稳定，建议使用 `PREFLIGHT_PROVIDER=mock` 准备录屏，真实 provider 作为可选展示。
