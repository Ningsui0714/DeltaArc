## 下一步门槛

要完成真实远端模型调用，需要在 `.env.local` 填入以下任一项：

```text
DOUBAO_API_KEY=<本机密钥>
```

或：

```text
ARK_API_KEY=<本机密钥>
```

若账号使用私有或受控 endpoint，还需要配置：

```text
DOUBAO_TEXT_ENDPOINT_ID=
DOUBAO_VISION_ENDPOINT_ID=
```
