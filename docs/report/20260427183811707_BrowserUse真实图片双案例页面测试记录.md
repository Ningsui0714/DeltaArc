## 结果

已使用 Browser Use in-app browser 完成两条 imagegen 真实图片案例的页面级测试。两个案例均在浏览器页面内完成图片载入、点击“开始传播推演”、结果展示校验，并保存全页截图。

## 背景处理

Browser Use 当前 runtime 不暴露 `setInputFiles`，不能直接把本地图片塞进 `<input type="file">`。为避免卡在系统文件选择框，新增了浏览器自动化专用 query 参数入口：

- `?preflightCase=imagegen-dorm`
- `?preflightCase=imagegen-campus-demo`

页面会从 `public/preflight-fixtures/` 读取 imagegen 图片并填入同一套页面状态，随后仍通过真实页面按钮触发推演。

- `public/preflight-fixtures/case-01-dorm-desk-cover-imagegen.png`
