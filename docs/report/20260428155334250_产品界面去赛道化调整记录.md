# 产品界面去赛道化调整记录

## 结果

已按反馈移除产品界面里的比赛感表达。当前 App 内不再出现 `第五赛道`、`Track 05`、`扣题`、`赛道` 等词，界面回到真实产品语气：`KOC Growth Lab`、`KOC Social Growth Agent`、`KOC 增长链路`、`发布风险护栏`。

## 改动

- `src/pages/PreflightStudioPage.tsx`
  - 首屏 eyebrow 从 `Track 05 / KOC Social Growth Agent` 改为 `KOC Social Growth Agent`。
  - 首屏副标题从“第五赛道作品...”改为“面向普通 KOC 的社媒 AI Agent...”。
  - 结果区标题从“第五赛道扣题链路”改为“KOC 增长链路”。
  - 隐藏内部提交分支里的“赛道匹配 / 同赛道参赛者”改为“方向匹配 / 同类参赛者”。

- `src/pages/PreflightStudioPage.test.tsx`
  - 测试新增断言：默认产品页面不得出现 `第五赛道 / Track 05 / 扣题`。

- `server/lib/preflight/prompt.ts`
  - prompt 内部说明从 fifth-track summary 改为 KOC growth summary，降低模型把赛道词带回用户结果的概率。

- `README.md`、`README.en.md`
  - 产品第一句话改为自然产品说明。
  - 参赛信息下沉到“参赛说明 / Competition Note”，明确赛道信息只属于提交材料、PDF 和录屏脚本，不进入产品 UI。

## 验证

- `npm run typecheck`：通过。
- `npm test`：通过，121/121。
- `npm run build`：通过。
- Browser Use 当前页面复查：
  - `KOC Growth Lab`：存在。
  - `KOC Social Growth Agent`：存在。
  - `社媒 AI Agent`：存在。
  - `第五赛道`：不存在。
  - `Track 05`：不存在。
  - `扣题`：不存在。
  - `赛道`：不存在。

## 决策

产品界面按真实 KOC 增长工具表达；比赛赛道只在提交材料、PDF、录屏脚本和说明文档中交代。
