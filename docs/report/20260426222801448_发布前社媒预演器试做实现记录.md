## 产品变化

旧版主体验是“生成一份分析报告”。新版主体验改为：

1. 上传或载入待发布作品。
2. 中间直接看到类似小红书的帖子预览。
3. 下方看到模拟互动数据和评论区。
4. 右侧看到 AI 的读片、推荐回复、涨粉动作和人群分层。

这样评委打开后能更快理解：这是一个“发布前社媒增长沙盘”，不是普通 AI 文案分析工具。

## 风险

- 当前手机预览是产品化视觉试做，不是小红书真实 UI 复刻。
- 一键示例默认使用占位封面；用户手动上传图片后会替换为真实图片。
- 本轮 Browser Use 推演使用本地 mock 服务验证交互闭环，真实豆包多模态链路仍沿用原接口。
