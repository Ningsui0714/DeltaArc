# 发布前试映场重构记录

## 本次交付

- 将主产品入口切换为“发布前试映场”
- 新增 `preflight simulation` 后端链路与异步 job API
- 新增 Doubao / 火山方舟多模态 provider
- 新增 mock provider，保证无 API key 环境也能跑通测试与 demo
- 新增 CLI harness、fixture、本地图片样例和 expected snapshot

## 主入口变化

- 前端默认页面不再是旧的内容风洞工作流，而是轻量发布编辑器
- 用户默认只需填写平台、目标、正文和图片
- 输出固定为：
  - 图片理解
  - 推流人群分布
  - 模拟首批回复
  - 风险与误解
  - 发布前修改建议

## 新 API

- `POST /api/preflight-simulations`
- `GET /api/preflight-simulations/:jobId`

## 新测试与回归入口

- `npm run typecheck`
- `npm test`
- `npm run build`
- `npm run harness:preflight`

## 说明

- 旧 sandbox / impact scan 代码保留在仓库中，作为兼容层和历史能力
- 新 preflight 主链路不再调用 DeepSeek
- Doubao 接口遵循火山方舟官方 OpenAI-compatible chat completions 文档
