# 小红书发布前试映样例

这些样例是给发布前试映场使用的内容稿，每个样例都配了封面图，并可同步接入 `?preflightCase=` 浏览器测试入口。

## 样例清单

- `qingyu-coffee-blind-test.request.json`：校园咖啡室友盲测，到店转化。
- `coop-game-night.request.json`：双人合作游戏试玩笔记，评论互动和涨粉。
- `yihuan-city-preview.request.json`：都市开放世界新品预热，认知曝光。
- `genshin-return-guide.request.json`：长线游戏回坑攻略，收藏和评论互动。
- `bigops-ecommerce-return-campaign.request.json`：电商大促返场运营，购买转化。
- `bigops-rain-delivery-notice.request.json`：本地生活雨天履约通知，认知与信任。
- `bigops-streaming-drama-reservation.request.json`：长视频新剧预约运营，评论互动。
- `bigops-ai-meeting-template.request.json`：企业效率工具 AI 模板上线，留资转化。

## 图片

图片放在 `images/` 下，同时复制到 `public/preflight-fixtures/`，用于前端 Browser Use 测试加载。`png` 是原始生成图，`jpg` 是真实 provider 测试使用的压缩版本。
